FAQ❓

What is Static url?
External site urls are called as static urls.
Ex: https://stockoverflow.com
What is Dynamic url?
Dynamic url contains just a url path.
It will combine with your current domain, when you click the urlbox.
Ex: path=tools/test/order/1 or tools/test/order.php, While click that box, you are in https://stockoverflow.com site, then new tab be like, https://stockoverflow.com/tools/test/order.php
What are these icons(⚙️, ⇉)?
These icons(⚙️, ⇉) are navigation icons.
It helps to you navigate one page from another page.
This icon (⚙️) navigate to settings page, This icon (⇉) navigate to view page.
How to open the urls?
View page: Just click the square box, which you want to open.
Settings page: Just press CTRL and click the square box, which you want to open.
How can I add Static & Dynamic urls?
Go to settings page.(click this ⚙️)
Click insert tool and fill the title and url or url path of your tool.
If url in url box start with http or https, then it will save as a External url. Other wise it will save as a Static url.
Click add button. Now you can see your title in below box.
